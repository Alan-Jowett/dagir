Detailed Description

Canonical IR attribute keys and backend interpretation guidance.

The constants in this header are the canonical keys used by policy objects and IR builders. Backends are expected to interpret these keys and map them into renderer-specific attribute names and semantics when producing output (GraphViz, JSON, Mermaid, etc.). The documentation below explains the intended meaning for each key and recommended backend behaviour.
