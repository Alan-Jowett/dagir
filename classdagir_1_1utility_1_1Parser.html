Detailed Description

Recursive descent parser for logical expressions.

Grammar (in order of precedence, lowest to highest): expression -> xor_expr xor_expr -> or_expr (XOR or_expr)* or_expr -> and_expr (OR and_expr)* and_expr -> not_expr (AND not_expr)* not_expr -> NOT not_expr | primary primary -> VARIABLE | ( expression )
