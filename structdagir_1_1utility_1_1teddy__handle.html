std::uint64_t dagir::utility::teddy_handle::stable_key ( ) const
inlinenoexcept

Stable key for memoization.

The key encodes the identity of the underlying TeDDy node using its pointer value. This function relies on the guarantee that node pointers remain stable for the lifetime of the view (i.e. the underlying diagram must not move or free nodes while the view or any memoization using this key is in use). Terminals are not encoded specially here; callers that need logical-terminal discrimination should handle that externally.
