Detailed Description

Header-only GraphViz DOT renderer for dagir::ir_graph.

This header provides a minimal, header-only helper to format a dagir::ir_graph as a GraphViz DOT digraph into an output stream. It honors a small set of dagir::ir_attrs where applicable (labels, colors, shapes, style hints) and performs conservative escaping of attribute values so the produced DOT is syntactically valid.

The implementation is intentionally small and header-only so it can be used by tests and tools without introducing additional build-time dependencies. Internal helpers live in the detail namespace; the primary public entry point is dagir::render_dot.
