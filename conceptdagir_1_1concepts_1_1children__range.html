Detailed Description

True if R is a range of edge_ref<..., H>-compatible elements.

Template Parameters
RThe range type to check (must satisfy std::ranges::input_range).
HThe handle type used by edges (for example a node identifier).

The concept accepts either a range whose value type models edge_ref<..., H> or a range whose reference type (after removing cvref) models edge_ref<..., H>. This allows it to match both owning ranges and ranges of references/views.
