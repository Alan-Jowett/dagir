Detailed Description

Concept for node attribute-producing policy callables.

Policies modeling dagir::concepts::node_attributor are callables that accept a read_only_dag_view and a node handle and return an attribute representation (commonly dagir::ir_attr_map). The return type is intentionally unconstrained to allow flexibility.
